# To-Do List

## Описание проекта
Мы разрабатываем backend-приложение для управления списком задач (To-Do List).  
Приложение позволит пользователям создавать, редактировать, просматривать и удалять задачи.

Проект реализуется на языке Python с использованием фреймворка FastAPI.

---

## Проблема
Многие люди сталкиваются с трудностями в организации своих задач: важные дела забываются, а список дел хранится в разных местах (заметки, мессенджеры, бумажные записи).

Это приводит к потере времени и снижению продуктивности.

---

## Цель проекта
Создать удобное API для управления списком задач, которое позволит:

- хранить задачи в одном месте
- отслеживать их статус
- удобно управлять списком дел

---

## Функциональность приложения
Планируется реализовать следующий функционал:

- создание новой задачи  
- просмотр списка всех задач  
- просмотр конкретной задачи по ID  
- редактирование задачи  
- удаление задачи  
- изменение статуса задачи (выполнена / не выполнена)
